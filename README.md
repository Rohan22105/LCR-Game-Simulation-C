# LCR Game Simulation in C
This project is a simulation of the Left-Center-Right (LCR) dice game written in C. The program simulates gameplay between multiple players using modular game logic, randomized dice rolls, and chip tracking.

## Features
- Simulates full LCR gameplay
- Supports multiple players
- Uses randomized dice rolls
- Tracks chip counts and player turns
- Modular program structure using multiple source files

## Skills Used
- C Programming
- Program Simulation
- Random Number Generation
- Modular Design
- Command-Line Interaction
- Makefiles

## Files
- `lcr.c` — Main gameplay loop
- `game.c` — Game logic functions
- `game.h` — Game definitions
- `randi.c` — Random number generation
- `randi.h` — Random utilities
- `strings.h` — Player names and strings
- `Makefile` — Build configuration

## Compilation

```bash
make
```

## Run

```bash
./lcr
```
